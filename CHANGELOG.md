## [0.1.0] - 2026-03-14

### Features

- Initial clay shooting application with personal training result tracking.
- Administrator user management.
- Club manager foundation with clubs, memberships, main-club selection, and a club-first home flow.
- Public club pages and dedicated club module pages for news, events, board information, and membership renewal.
- Admin publishing screens for club content and renewal settings.
- Member-facing renewal request workflow with admin review.

### Documentation

- Initial project setup and test environment documentation.
